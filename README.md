## 🛠️ Troubleshooting Common Issues

**Game crashes or doesn't start:**  
- Verify that GTA V and mods like FiveM are updated.  
- Check that you placed files in the correct folders.  
- Disable other reshade or graphic mods that might conflict.

**Visual effects not showing:**  
- Make sure reshade is enabled in the game's options menu.  
- Confirm you launched the game through the proper mod client (FiveM, RageMP, altV).

**Performance drops:**  
- Lower reshade shader settings in the menu.  
- Avoid running other screen effect software simultaneously.
